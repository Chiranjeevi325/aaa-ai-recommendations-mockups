Acceptance criteria

  • ✓ Only Advanced tier students see AI recommendations
  • ✓ Free / Standard see an upgrade prompt
  • ✓ Recommendations appear on dashboard and after tests
  • ✓ Minimum 5 tests before recommendations activate
  • ✓ Refresh daily + after each test
  • ✓ Difficulty adapts (>80% = harder, <50% = easier)
  • ✓ Basic ML model (not deep learning)
  • ✓ Recommendations load within 2 seconds

Out of scope (this phase)

  • — Daily study plan (deferred)
  • — Forgetting curve / spaced repetition
  • — Advanced ML / deep learning models
  • — Real-time ML inference (batch is sufficient)

Dependencies

  • P2-01 Video Player · P2-02 Test Interface
  • P2-03 Results & Feedback · P2-04 Dashboard
